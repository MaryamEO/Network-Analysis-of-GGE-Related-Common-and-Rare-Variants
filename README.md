# Network-Analysis-of-GGE-Related-Common-and-Rare-Variants
A workflow to define the contributions of both common and rare variants to Genetic Generalized Epilepsies (GGEs) at the molecular network level.

# Dependencies
Here we have listed the software that have been employed for the stated analysis 


| Software   | Purpose |  
| ------------- | ------------- |
| [NetColoc](https://pypi.org/project/netcoloc/)|Building colocalized gene networks|
| [UpSetR ](https://github.com/hms-dbmi/UpSetR)  |Defining the intersection of the gene sets|
