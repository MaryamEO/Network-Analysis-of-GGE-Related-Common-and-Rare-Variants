# Network-Analysis-of-GGE-Related-Common-and-Ultra-Rare-Variants
A workflow to define the contributions of both common and ultra rare variants to Genetic Generalized Epilepsies (GGEs) at the molecular network level.

# Dependencies
Here we have listed the software that have been employed for the stated analysis
